# Projeto Semana OminiStack

Projeto feito baseado na semana oministack 10 pela Rocketseat.

## Sobre

Esse projeto tem o objetivo de criar uma aplicação para desenvolvedores inspirada no uber.

## Tecnologias usadas

* **Express** - Backend
* **MongoDb** - DataBase
* **React** - Web
* **React Native** - Mobile
